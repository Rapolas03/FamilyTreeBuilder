Usage Instructions

Add Person: Type a name and click "Add Person"
Select Children: Click on person(s) to select as children (green border)
Select Parents: Hold Ctrl + Click on 1-2 people to select as parents (gold border)
Connect: Click "Connect" to create relationships
Move Nodes: Drag any person to reposition them
Remove: Select person(s) and click "Remove Person"
